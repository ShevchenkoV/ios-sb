Entering test administration details

Begin by entering some basic details of the testing session, including:

An examinee or patient identifier - this may include numbers or letters.

Examinee DOB – using the calendar function.

Examiner initials.

Date of testing – using the calendar function. This will default to the current date, but can be modified if required.

Completing subtests

When completing the SYDBAT, Naming is always administered first. In addition, Comprehension needs to be administered before Semantic Association. If possible, provide a delay (~15 min) between administering Naming and Comprehension tests.

Select each subtest by pressing the button relating to the name of the test. The administration script will appear. After this page, place the iPad or tablet in front of the examinee to view the stimuli.

For the Naming and Repetition subtests, an audio recording commences for each item from the time the picture is displayed until the response is scored or the examiner advances the test to the next item [by swiping left with one finger].

For the Comprehension and Semantic Association subtests, the examinee indicates his/her answer by pressing on the corresponding picture. This will produce a red box around the image and will expand the image in size, for the examinee to confirm the response. Once the examinee has indicated this is the final response, the examiner swipes left with one finger to continue.

Scoring gestures

To score a response as “correct”, perform a short swipe to the left of the screen using one finger.

To score a response as “incorrect”, perform a long swipe to the left of the screen using one finger.

To end the test once 6 consecutive errors are recorded (discontinue rule), swipe up or down on the screen using one finger.

To abort a test before completion, swipe up or down on the screen using two fingers.

Correct, Score = 1
Incorrect, Score = 0

Reviewing responses

In some instances, it may be necessary to review a response before confirming the final score (e.g. on the Naming or Repetition subtests if a response is unclear and you wish to listen to the recording again before assigning a score). To do this, simply go to the View Activity Log.

Select the specific test needing review: listed by test name, ID and testing date.

Select the item you wish to review. This will open a new window that will allow you to change the score and add comments.

Exporting results

Once the test is completed and you wish to export the results:

Open ‘View Activity Log’ on the home screen.

Select the specific test for exporting: listed by test name, ID and testing date.

In the top right corner, select ‘export to email’.

Choose the preferred Email App from the list provided in the pop-up window.

Note: you may need to configure an email account with your tablet in order to export data via email.
