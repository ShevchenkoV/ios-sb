Naming

Introduce the test with the script provided. For each item displayed, repeat the question "what is this called?" until the examinee is familiar with the instruction.

Feedback should be provided on each of the practice items

If correct: That's right, this is a [kangaroo / strawberry]

If incorrect: This would be called a [kangaroo / strawberry]

No feedback should be given on answers for the test items.

The following prompts may be given during the subtest:

If an abbreviated version of the word is given (e.g. "bike", "PC"), then ask "what is the full name?"

The response is vague or a description of the item is given (e.g. "oh that's a type of food"), then ask "can you give me the exact name?"

If an alternative or colloquial version is given (e.g. "wireless"), then ask "do you know another word for that?" or "what is the formal name for that?"

If the person grossly misperceives an item (e.g. Item 25 - tiara, some participants do not recognise that this is sitting on top of someone's head), then clarify (e.g. "this is on someone's head").

If for Item 28, the person says "robber", clarify, "but what is the name of the thing he is wearing?"

Where feasible, administer all items of the test. If however, the examinee has failed more than 6 consecutive items and appears distressed, then discontinue the test by using a one-finger swipe up or down. The examiner is alerted that 6 consecutive items have been failed when the item number is highlighted with a grey box.

For each item, use the gestures to indicate whether the responses are correct, incorrect, or uncertain. Responses are audio recorded and can be reviewed after testing via the Activity Log, if required.

Repetition

Introduce the test with the script provided. For each item, play the sound recording, and if required repeat the question "Now you say...[word]".

Common alternative pronunciations are accepted, however where possible, the person should be encouraged to model the pronunciation provided in the recorded version of each word.

For practice items, it is not necessary for the person to be able to correctly repeat the word. The practice items are to ensure understanding of the task instruction.

The first attempt to repeat the word is the response scored. If there is no attempt to say the word, or if before attempting to say the word the examinee asks for the word again, the word can be replayed.

Administer all items of the test. However, you may wish to discontinue after 6 consecutive errors if the examinee appears distressed.

For each item, use the gestures to indicate whether the responses are correct, incorrect, or uncertain. Responses are audio recorded and can be reviewed after testing via the Activity Log, if required.

Word comprehension

Introduce the test with the script provided. For each item, ask "Where is the..." and play the sound recording. Once the examinee is familiar with the instruction, simply play the sound recording when displaying the item.

Feedback should be provided on each of the practice items

If correct: That's right, this is a [kangaroo / strawberry]

If incorrect, point to the correct item and say: This is the best answer for [kangaroo / strawberry]


No feedback should be given on answers for the test items.

The sound recording may be repeated as many times as necessary.

Once a response has been selected, the picture will be highlighted and will expand in size, so that the examinee can review the selection. If necessary, the examinee can then select another response, without penalty. Confirm when the examinee is satisfied with their response, and then swipe left with one-finger to advance to the next item. After advancing to the next item, it is not possible for the examinee to go back and select another choice. If the person begins naming all the items on the page say: "yes, but where is the [repeat word]?"

Where feasible, administer all items of the test. If however, the examinee has failed more than 6 consecutive items and appears distressed, then discontinue the test by using a one-finger swipe up or down. The examiner is alerted that 6 consecutive items have been failed when the item number is highlighted with a grey box.

Semantic Association

Introduce the test with the script provided. For each item, ask "Which one best goes with the [name the item]". Once the examinee is familiar with the instruction, simply display the item.

Feedback should be provided on each of the practice items

P1 if correct: That's right, the kangaroo goes with the Opera House because they are both Australian things

P1 if incorrect: The best answer would be the Opera House because they are both Australian things

P2 if correct: That's right, the strawberry goes with the cream because many people like to eat strawberries with cream

P2 if incorrect: The best answer would be the cream because many people like to eat strawberries with cream

No feedback should be given on answers for the test items.

Once a response has been selected, the picture will be highlighted and will expand in size, so that the examinee can review the selection. If necessary, the examinee can then select another response, without penalty. Confirm when the examinee is satisfied with their response, and then swipe to the left with one-finger to advance to the next item. After advancing to the next item, it is not possible for the examinee to go back and select another choice.

If the person points only to the top picture: "yes, but which one of these [motion to the 4 pictures below] does it go with?"

Where feasible, administer all items of the test. If however, the examinee has failed more than 6 consecutive items and appears distressed, then discontinue the test by using a one-finger swipe up or down. The examiner is alerted that 6 consecutive items have been failed when the item number is highlighted with a grey box.

Naming

Score as correct if:

The examinee clearly says the correct name. This includes instances where the examinee self-corrects, without any feedback (e.g. "television...uh..no..computer!").


The examinee produces all components of the word said, but with some slight distortion of phonemes (e.g. "stesascope" instead of stethoscope would be accepted; "stemascope" would be incorrect).

Score as incorrect if:

The examinee gives a semantically related response (e.g. "letter" for envelope).

Not all components of the word are said (e.g. "envel..." for envelope or "rh-n-sis" for rhinoceros).

The examinee gives a word approximation (e.g. "stemascope" for stethoscope or "hippopegos" for hippopotamus).

They were not administered due to the test being discontinued after 6 consecutive failures.

Items are ideally scored by the examiner during the administration of the test (i.e. when presenting each item). A temporary "uncertain" option can, however, be selected if the examiner is unsure and needs to review the responses after testing is complete.

Repetition

Score as correct if:

All syllables are correctly pronounced and are done so without pauses between syllables/uneven delivery of syllables in a word, distortion in phonemes or clusters (e.g. "fl" or "gl"), or restarting due to articulation problems (indicating laboured speech e.g. stuttering on the first sound, self-corrected sounds).


The examinee adds an "-s" to the end of the word or leaves off an "-s" at the end of the word. Other common, learned pronunciations of the word are accepted as long as they conform to the above rules. In examinees who have English as a second language, minor variations due to accent are acceptable (e.g. hippopotamus = 'ippoptamus).

Score as incorrect if:

The examinee cannot say the word.

The examinee must re-start the word in order to say it correctly (e.g. stuttering over first phoneme).

The examinee uses the wrong phoneme in the word (e.g. "dande-iron" for dandelion).

The examinee clearly segments the word or says some syllables slowly and others in a rush (e.g. "hip-po-pot-a-mus"; or "hip-po-potamus").

Items are ideally scored by the examiner during the administration of the test (i.e. when presenting each item). A temporary "uncertain" option can, however, be selected if the examiner is unsure and needs to review the responses after testing is complete.

Word Comprehension

Responses to the Word Comprehension and Semantic Association subtests are automatically scored through the app.

The examinee is scored as correct if the correct picture is selected (even if their reasoning is incorrect). Only the final selection made for each item is scored (i.e. the examinee may initially select the wrong picture but if this is self-corrected, without feedback, then the response is considered correct).

Any items which are not administered due to the test being discontinued after 6 consecutive failures are considered incorrect.
